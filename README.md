# 🧠 Deepfake Image Detection System

This project detects whether an image is REAL or FAKE using a deep learning model.

## 🚀 Features
- Upload image and detect deepfake
- Shows real vs fake probability
- Provides explanation of prediction
- Visual feature analysis (heatmap)

## 🛠 Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Streamlit

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## 📊 Output
- Real vs Fake prediction
- Confidence percentage
- Feature-based explanation
